# GitHub Labels Documentation

Below are standard labels configured for the PRAHARI repository:

| Label | Color | Description |
|---|---|---|
| `bug` | `#d73a4a` | Something isn't working as expected |
| `enhancement` | `#a2eeef` | New feature request or improvement proposal |
| `documentation` | `#0075ca` | Changes to documentation or guidelines |
| `duplicate` | `#cfd3d7` | Issue already reported |
| `wontfix` | `#ffffff` | Issue will not be addressed |
| `good first issue` | `#7057ff` | Suitable task for new contributors |
| `help wanted` | `#008672` | Core team requires community help |
| `security` | `#e11d21` | Vulnerability warning or compliance update |
