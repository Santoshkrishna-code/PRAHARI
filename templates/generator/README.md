# PRAHARI Developer Platform CLI Generator

This tool scaffolds services, workers, and cronjobs conforming to PRAHARI standards.

---

## 1. Commands

- **`init`**: Registers local registry catalog.
- **`new service <name>`**: Starts code scaffolding.
- **`validate`**: Asserts clean architecture boundaries rules.
- **`doctor`**: Verifies local environment.

---

## 2. CLI Examples

```bash
prahari new service incident
```
