# Security Policy

## Supported Versions

Only the latest `main` branch version is supported with security updates.

---

## Reporting a Vulnerability

If you identify a security issue in the PRAHARI platform, do not log a public GitHub Issue. Instead:

1. Draft a descriptive explanation of the vulnerability and steps to reproduce.
2. Email the report privately to `security@prahari-ehs.internal`.
3. The security triage team will respond within 48 hours to confirm and assign fix targets.
